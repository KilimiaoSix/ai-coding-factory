# 08 实现任务

## 任务列表

1. 建立 `AgentProfile` 配置模型
2. 建立证据索引链
3. 建立 transcript 引用链
4. 建立关键状态审计链

## 任务拆分

| 任务 | 输入 | 输出 |
| --- | --- | --- |
| `8.1` 实现 `AgentProfile` | 角色和模型配置需求 | 控制平面配置对象 |
| `8.2` 建立证据索引 | `Evidence` 与 artifact 输出 | 统一证据链 |
| `8.3` 建立 transcript 引用 | runtime transcript | 对象可追溯引用 |
| `8.4` 建立审计事件记录 | phase、Gate、bug 等状态变化 | 审计链与查询接口 |

## 完成定义

- 角色配置可在控制平面中查看
- 关键证据和 transcript 可被正式对象引用
- 关键状态变更可追溯
