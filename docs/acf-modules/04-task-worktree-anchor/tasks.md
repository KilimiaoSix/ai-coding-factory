# 04 实现任务

## 任务列表

1. 建立 `Task` 与 `TaskRun` 生命周期
2. 建立写任务隔离策略
3. 建立 `code_anchor_ref` 规则
4. 建立执行归因链

## 任务拆分

| 任务 | 输入 | 输出 |
| --- | --- | --- |
| `4.1` 实现 `Task/TaskRun` 流程 | 对象模型 | 任务执行链 |
| `4.2` 实现 worktree 分配规则 | 任务类型与角色配置 | 隔离策略和 worktree 引用 |
| `4.3` 实现代码锚点捕获 | git/worktree 状态 | `code_anchor_ref` |
| `4.4` 建立任务归因接口 | runtime 结果与对象关系 | 任务执行摘要和引用链 |

## 完成定义

- 写任务可默认隔离
- 每次执行有正式实例
- 代码版本可被后续验证和 Gate 引用
