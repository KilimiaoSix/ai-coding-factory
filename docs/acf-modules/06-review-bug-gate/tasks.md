# 06 实现任务

## 任务列表

1. 建立正式评审链
2. 建立缺陷回流机制
3. 建立阻塞升级链
4. 建立 Gate 输入检查与裁决

## 任务拆分

| 任务 | 输入 | 输出 |
| --- | --- | --- |
| `6.1` 实现 `ReviewRecord` | 阶段评审规则 | 正式评审对象 |
| `6.2` 实现 `Bug` 流程 | `VerificationRun` 结果 | 缺陷创建、修复、复测流程 |
| `6.3` 实现 `Blocker` 流程 | 执行阻塞和用户反馈 | 阻塞升级与解决链 |
| `6.4` 实现 `GateDecision` | Gate 规则与输入 | 阶段裁决与状态推进 |

## 完成定义

- 阶段评审有正式对象
- 缺陷与阻塞分别有闭环
- Gate 推进不再依赖隐式聊天判断
