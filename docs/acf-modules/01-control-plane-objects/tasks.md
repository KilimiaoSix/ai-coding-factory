# 01 实现任务

## 任务列表

1. 定义首期对象 schema
2. 定义统一枚举和状态机约束
3. 初始化本地数据库基线
4. 建立对象间引用与索引
5. 建立 Artifact 引用层

## 任务拆分

| 任务 | 输入 | 输出 |
| --- | --- | --- |
| `1.1` 定义对象类型 | 蓝图第 6 章 | 统一类型定义 |
| `1.2` 定义状态枚举 | 蓝图第 6.8-6.11 节 | 枚举与校验规则 |
| `1.3` 初始化存储基线 | 对象清单与字段表 | 本地数据库和迁移基线 |
| `1.4` 建立对象仓储接口 | schema 与存储 | repository/service 接口 |
| `1.5` 建立 Artifact 引用策略 | `Evidence` 和 transcript 约束 | 证据索引与引用工具 |

## 完成定义

- 所有首期对象有统一 schema
- 所有关键状态有枚举约束
- 本地存储可初始化
- 后续模块可通过对象接口编程
