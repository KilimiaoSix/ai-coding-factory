# Claude Code Handbook

本目录是基于当前仓库源码快照整理的中文项目手册。它以 `../other-ans-archive/` 的 45 章分析稿为主骨架，结合同级参考目录下 `00-09` 模块说明和当前 `src/` 目录的真实结构，重写为一套可以连续阅读、也可以按章节跳读的项目说明。

## 手册定位

- 目标不是替代源码，而是解释这份 TypeScript 快照如何组织成一个可运行的终端 Agent。
- 内容优先回答“为什么这样设计”和“当前快照里这一层具体落在哪些文件”。
- `../other-ans-archive/` 更像长篇分析草稿；本目录则是校正后的正式手册。

## 源码快照边界

- 本仓库是一个研究用途的外部快照，不包含全部内部 Anthropic 包。
- 许多能力受 `feature()` 门控，外部构建默认关闭，见 [scripts/build.ts](../../scripts/build.ts)。
- 一些路径存在桩实现或仅用于内部发行版，文中会统一标注为“门控/缺失/外部快照不可完整运行”。

## 阅读方式

- 先看 [SUMMARY.md](./SUMMARY.md) 了解 12 个部分和 45 章结构。
- 再看 [source-map.md](./source-map.md) 理解“现有模块文档、章节、源码目录”的映射关系。
- 如果你只关心主链路，建议按 `ch01` → `ch05` → `ch07` → `ch14` → `ch20` → `ch31` → `ch43` 阅读。

## 与原有 docs 的关系

- 同级 `00-09` 模块文档：偏模块速查，适合快速定位。
- `../other-ans-archive/*.md`：偏叙事分析，覆盖范围更广。
- `claude-code-handbook/*.md`：把两者合并为统一术语、统一结构、统一快照说明的正式手册。

## 写作约定

- 章节编号与标题沿用 `other-ans` 的 45 章结构。
- 每章默认包含：问题定义、架构分析、关键源码锚点、快照修正与补充、设计启示。
- 文中路径指向当前源码快照，不使用推测性的内部实现名称。
